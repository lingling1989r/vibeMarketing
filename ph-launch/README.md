## 📋 Pre-Launch Checklist

### PH Profile
- [ ] Maker profile complete (avatar, bio, links)
- [ ] At least 5 previous products supported/upvoted (builds credibility)
- [ ] Twitter/X linked to PH profile

### Listing
- [ ] Tagline finalized (< 60 chars) → see `en/listing.md`
- [ ] Description written (250-500 words)
- [ ] 3-5 screenshots ready (1270×952 recommended)
- [ ] Demo video (60s max, no music needed — captions instead)
- [ ] First Comment drafted → see `en/first-comment.md`
- [ ] Maker Comment templates ready for rapid response

### Outreach
- [ ] 3-5 hunters confirmed (ideally 500+ followers)
- [ ] 20+ friends/users ready to upvote at launch
- [ ] Twitter audience warmed up (5+ podcast-related tweets in past 2 weeks)
- [ ] IH post scheduled for launch day

### Technical
- [ ] Website handles traffic spike (test load)
- [ ] English landing page optimized for PH audience
- [ ] Sign-up flow works perfectly (test 5x)
- [ ] Special PH welcome page or discount code ready
